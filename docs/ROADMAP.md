# Roadmap

## MVP

- Go HTTP 服务端。
- 图片帧识别接口。
- Mock 识别器。
- 内存缓存。
- 返回英文字母标签、单词、释义、音标、例句。

## V1

- 接入云端视觉 API。
- 增加图片压缩和上传限流。
- 增加持久化缓存。
- 增加设备 ID、用户学习历史。
- 增加 WebSocket 推送识别结果。

## V2

- 局部变化检测。
- 相似场景 embedding 缓存。
- 常见物体本地模型。
- 离线模式。
- 单词复习和错词本。

## V3

- 自训练检测模型。
- ONNX/NCNN/TensorRT 边缘推理。
- 摄像头运动预测和预加载。
- 多语言学习模式。
